# QiYuan's Web — Jekyll/Hyde on GitHub Pages

This is QiYuan's personal website built with Jekyll and the Hyde theme.

## Content Editing

When creating, editing, or managing content (posts, blogs, figures, shots, vlogs), always use the `qyweb_editing` skill. It contains all site conventions, templates, and procedures.

## Content Collections

- `_posts/` — Technical notes and learning entries
- `_blogs/` — Research/project blog updates
- `_figs/` — Art and figure galleries
- `_shots/` — Photo shot galleries
- `_vlogs/` — Video log entries (Bilibili embeds)
